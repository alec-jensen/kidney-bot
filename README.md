# kidney-bot
this is all the code that makes the bot go.
there is 4 main folders:
- bot folder. this contains all the discord.py code that runs the actual bot.
- api folder. this contains all the fastapi code for the main api (relies on some apis also implemented in the bot)
- website folder. this contains all the website files
- dashboard folder. this contains all the dashboard files
